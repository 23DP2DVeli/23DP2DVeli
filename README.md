## Projects

**MovieClub** — Full-stack movie catalogue platform

A personal project built during my internship. Users can browse films, read descriptions, leave comments, and rate movies. Admins can manage content through a protected panel.

- Laravel REST API with Sanctum authentication and role-based access
- React + TypeScript frontend with Zustand state management
- Docker-based deployment on AWS Lightsail via Bitbucket CI/CD pipelines
- OpenAPI documentation generated automatically with Scramble

[View on Bitbucket](https://bitbucket.org/movieclub1/workspace/projects/MOV2)
